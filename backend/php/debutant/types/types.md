**NB : Pour tous les exercices vous travaillerez avec une page php sur votre serveur qui vous servira à afficher les résultats obtenus.**

## 1\.Exercice 1
Ecrivez la fonction PHP qui permet d'afficher quelque chose à l'écran et terminez votre instruction par un point virgule.
Voyez-vous quelque chose s'afficher ? Pourquoi ?

Ecrivez la fonction PHP qui permet d'afficher quelque chose suivie de votre âge.
Voyez-vous quelque chose s'afficher ? Pourquoi ?

Quel est le type des données que vous venez d'afficher ?

## 2\.Exercice 2
Affichez à l'écran le texte suivant en respectant scrupuleusement la notation : "Hello World".

Quel est le type des données que vous venez d'afficher ?

Que se passe-t-il si vous retirez les guillemets autour de Hello World ? Pourquoi ?

Que se passe-t-il si vous remplacer les guillemets doubles par des guillemets simples '' ?

Que se passe-t-il si vous oubliez de fermer vos guillemets comme ceci 'Hello World ? Pourquoi ?

## 4\.Exercice 4

Affichez à l'écran 45.
Affichez à l'écran 45.12
S'agit-il du même type de données ? Si non quelle est la différence ?

Que se passe-t-il si vous utilisez la virgule au lieu du point et affichez 45,12 ? Pourquoi l'ordinateur affiche-t-il ce résultat ? Astuce : regardez du côté de la documentation de echo
